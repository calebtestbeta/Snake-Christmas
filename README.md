# 文字貪食蛇 - 聖誕祝福版 🎄

一款融合台灣基督教信仰文化的創新文字貪食蛇遊戲，以聖誕節和基督教價值為主題，讓你在遊戲中收集祝福、體驗屬靈成長的樂趣！

## 🎮 遊戲特色

- **聖誕信仰主題**：收集各種聖誕與基督教信仰字符，從「聖誕快樂」到「哈利路亞」
- **詞句組合系統**：收集字符組成完整詞句，獲得特殊祝福效果
- **屬靈成長機制**：不同字符提供不同屬靈成長指標（信心、愛心、盼望、平安、喜樂、讚美）
- **動態效果**：字符會影響蛇的移動速度和行為，體現屬靈意義
- **三種難度**：平安夜、聖誕晨、報佳音模式，適合不同水平玩家
- **響應式設計**：完美適配桌面、平板和手機設備
- **屬靈分析**：遊戲結束後提供詳細的屬靈成長報告和聖經經文祝福

## 🚀 安裝與運行

### 環境需求
- 現代網頁瀏覽器（Chrome、Firefox、Safari、Edge）
- 支持 HTML5 Canvas 和 ES6+ JavaScript

### 運行方式

**方式一：直接開啟**
1. 下載或克隆此專案
2. 直接用瀏覽器開啟 `index.html` 文件

**方式二：本地服務器**
```bash
# 使用 Python（推薦）
python3 -m http.server 8000

# 或使用 Node.js
npx http-server

# 或使用 PHP
php -S localhost:8000
```

然後在瀏覽器中訪問 `http://localhost:8000`

## 🕹️ 遊戲操作

### 控制方式
- **鍵盤**：使用方向鍵 ↑↓←→ 控制蛇的移動
- **觸控**：點擊螢幕下方的虛擬方向按鈕
- **暫停**：按 `P` 鍵暫停/繼續遊戲

### 遊戲規則
1. **目標**：在 60 秒內收集聖誕與信仰字符，組成美好詞句
2. **移動**：控制蛇頭移動，避免撞牆或撞到自己
3. **成長**：收集字符後蛇身會變長，並獲得屬靈成長
4. **詞句獎勵**：組成完整詞句可獲得特殊效果
   - 🎄 「聖誕快樂」：金色光芒效果
   - ✝️ 「耶穌愛我」：穩定移動速度
   - 🙏 「哈利路亞」：雙倍字符出現
   - ⭐ 「以馬內利」：延長遊戲時間

### 字符分類與效果

#### 📿 信仰核心類（金色系）
- **字符**：主、神、耶、穌、愛、信
- **效果**：穩定持久的力量，建立堅固的信心根基
- **屬靈成長**：主要提升信心與愛心指標

#### ⭐ 聖誕慶典類（亮黃系）
- **字符**：聖、誕、快、樂、夜、音
- **效果**：歡樂的能量爆發，帶來節慶的喜悅
- **屬靈成長**：主要提升喜樂與盼望指標

#### 🎁 祝福話語類（紅色系）
- **字符**：平、安、福、恩、典、惠
- **效果**：溫暖的祝福力量，帶來心靈的安慰
- **屬靈成長**：主要提升平安與信心指標

#### 🕊️ 讚美敬拜類（白銀系）
- **字符**：哈、利、路、亞、讚、美
- **效果**：屬靈的升華，讚美中得著力量
- **屬靈成長**：主要提升讚美與喜樂指標

#### ❤️ 愛的分享類（粉紅系）
- **字符**：分、享、溫、暖、人、心
- **效果**：溫馨的情感力量，在愛中彼此建造
- **屬靈成長**：主要提升愛心與平安指標

### 可組成的詞句
