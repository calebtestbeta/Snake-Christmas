# 🎬 聖誕貪食蛇遊戲示範影片錄製指導

## 📋 錄製規格要求

### 技術規格
- **解析度**: 1280x720 (推薦) 或 1920x1080
- **時長**: 60-90 秒
- **檔案格式**: MP4 (H.264 編碼)
- **檔案大小**: < 20MB (目標 10-15MB)
- **幀率**: 30fps
- **音訊**: 可選（如需要可加背景音樂或旁白）

### 錄製工具建議
- **Mac**: QuickTime Player (內建) 或 ScreenFlow
- **Windows**: OBS Studio (免費) 或 Camtasia
- **線上工具**: Loom 或 CloudApp

## 🎯 錄製內容腳本

### 場景 1: 遊戲開始 (0-10秒)
1. 顯示遊戲起始畫面
2. 點擊「開始聖誕祝福之旅」按鈕
3. 展示倒數畫面 (3-5秒倒數)

### 場景 2: 基礎操作展示 (10-25秒)
1. 蛇開始移動
2. 使用方向鍵控制蛇的方向
3. 收集第一個聖誕字符
4. 展示蛇身變長和字符顯示

### 場景 3: 特色功能展示 (25-50秒)
1. 連續收集多個字符 (目標: 聖、誕、快、樂)
2. 展示蛇頭顏色變化效果
3. 展示右上角詞句進度提示
4. 收集到「聖誕快樂」字符組合

### 場景 4: 完成效果 (50-60秒)
1. 展示完成「聖誕快樂」詞句的慶祝特效
2. 顯示伯利恆之星特殊效果
3. 展示詞句完成提示訊息

## 📱 錄製注意事項

### 瀏覽器設定
- 使用 Chrome 或 Safari 最新版本
- 確保頁面 100% 縮放比例
- 關閉瀏覽器開發者工具
- 隱藏書籤列以獲得更大錄製區域

### 遊戲設定
- 選擇「平安夜」難度 (速度較慢，適合錄製)
- 確保所有聖誕視覺效果正常顯示
- 準備好操作流程，避免錄製中出錯

### 錄製技巧
- 錄製前先練習 2-3 次確保流暢
- 保持穩定的操作節奏
- 避免長時間停頓或猶豫
- 確保重要時刻 (字符收集、特效) 清晰可見

## 🎨 後期處理建議

### 影片編輯
- 裁剪掉開頭結尾的多餘畫面
- 確保影片流暢無卡頓
- 可加入簡單的片頭片尾 (3-5秒)

### 壓縮優化
- 使用 H.264 編碼器
- 位元率設定: 2000-4000 kbps
- 如超過 20MB，適當降低解析度至 1280x720

### 品質檢查
- 確保字符清晰可讀
- 檢查顏色對比度充足
- 驗證在不同設備上的播放效果

## 📂 檔案命名與格式

### 主要檔案
- `christmas-snake-demo.mp4` (主要格式)
- `christmas-snake-demo.webm` (備用格式，可選)

### 儲存位置
- 直接放在專案根目錄
- 確保檔案名稱與程式碼中的引用一致
